# Периметр

**Периметр** — интерактивная текстовая хоррор-игра для Telegram.

Игрок не вводит свободный текст. На каждом этапе он выбирает один из трёх вариантов ответа:

- 🟦 рациональный;
- 🟥 импульсивный;
- 🟩 эмпатичный.

Каждый выбор влияет на отношения, знания, мораль, страх и финал.

## Сеттинг

Заброшенный научный центр **«Периметр»**. В прошлом здесь изучали проект **«Эхо» / PX-17** — нейроактивное вещество, которое не создаёт монстров напрямую, а усиливает память, вину, страх и субъективное восприятие человека.

Главная идея игры: страшен не монстр, а момент, когда человек перестаёт доверять собственным чувствам.
