# Периметр

**«Периметр»** — интерактивная Telegram-игра в жанре психологического хоррора и научного триллера.

Игрок входит в заброшенный НИИ, где раньше изучали проект **«Эхо» / PX-17**. Препарат не создаёт монстров напрямую: он усиливает память, страх, вину и заставляет человека достраивать реальность собственными ожиданиями.

## Что внутри

- 530 сцен: Акт I — 120, Акт II — 180, Акт III — 230.
- Акт II закрыт, пока не пройден Акт I.
- Акт III закрыт, пока не пройден Акт II.
- Выборы без видимой цены: игрок не видит, что он теряет.
- Скрытые переменные: доверие NPC, страх, заражение PX-17, знание, моральный вектор.
- Психопрофиль перед входом.
- Журнал, кодекс документов, карта объекта, настройки, админ-команды.
- Картинки генерируются автоматически при первом запуске, поэтому проект удобно заливать на GitHub и Amvera без тяжёлых PNG-файлов.

## Быстрый запуск локально

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
```

В `.env` вставьте токен бота:

```env
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_IDS=8196658213
DB_PATH=data/perimetr.sqlite3
```

Запуск:

```bash
python app.py
```

## Запуск на Amvera

В Amvera добавьте переменные окружения:

```text
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_IDS=8196658213
DB_PATH=/data/perimetr.sqlite3
PORT=8080
PYTHONUNBUFFERED=1
```

Файл `amvera.yaml` уже лежит в корне. Точка входа — `app.py`.

## Команды игрока

- `/start` — стартовый экран.
- `/menu` — главное меню.
- `/new` — новая вылазка.
- `/continue` — продолжить.
- `/acts` — выбор акта.
- `/journal` — журнал.
- `/codex` — документы.
- `/map` — карта объекта.
- `/status` — состояние прохождения без чисел.
- `/settings` — настройки отображения.
- `/note текст` — личная заметка.
- `/theory текст` — гипотеза игрока.
