# Map System

## Идея

Карта комплекса не доступна игроку полностью с самого начала. Она открывается фрагментами через найденные предметы:

- **Карта первого этажа** — открывает КПП, двор, гараж, медпункт, холл, администрацию, архив, лаборатории и серверную.
- **Карта технического уровня** — открывает генераторную, электрощитовую, склад химии, вентиляцию и тоннели.
- **Карта подземного уровня** — открывает хранилище PX-17, испытательные камеры, центральную лабораторию, командный пункт, старый бункер и аварийный выход.

До нахождения карты игрок видит только текущую локацию и короткое описание направления. После нахождения карты соответствующий уровень появляется в мини-карте.

## Правило открытия

```text
Игрок нашёл карту первого этажа → /map показывает первый этаж.
Игрок нашёл карту технического уровня → /map показывает первый этаж + технический уровень.
Игрок нашёл карту подземного уровня → /map показывает все найденные уровни.
```

## Где находятся карты

### Карта первого этажа

**Локация:** главный холл, стойка регистрации.  
**Сцена:** после первого осмотра журнала посещений.  
**Сюжетная функция:** игрок понимает структуру комплекса и получает первые безопасные маршруты.

### Карта технического уровня

**Локация:** электрощитовая / генераторная.  
**Сцена:** после решения проблемы с питанием.  
**Сюжетная функция:** игрок видит, как связаны вентиляция, генератор и распространение PX-17.

### Карта подземного уровня

**Локация:** архив уничтоженных исследований или кабинет директора.  
**Сцена:** ближе к midpoint.  
**Сюжетная функция:** игрок понимает, что «Периметр» намного больше, чем казалось.

## В Telegram-боте

Команда:

```text
/map
```

Если игрок ещё не нашёл карты:

```text
У тебя нет карты комплекса. Сейчас ты ориентируешься только по памяти и указателям.
```

Если найдена карта первого этажа:

```text
Открыта карта: Этаж 1.
```

Если найдены несколько карт:

```text
Открыты карты: Этаж 1, Технический уровень, Подземный уровень.
```

## Интеграция с Web Map

HTML-карта лежит в папке:

```text
web_map/index.html
```

Она принимает параметры:

```text
?maps=floor1
?maps=floor1,tech
?maps=floor1,tech,underground
```

Telegram-бот формирует ссылку автоматически по найденным картам игрока.

## Атмосферное правило

Карта не должна выглядеть как чистый UI. Она должна быть внутриигровым документом: грязная бумага, красные пометки, следы влаги, служебные штампы и предупреждения. Стиль должен соответствовать постеру «Периметр»: тёмный, индустриальный, архивный, тревожный.
