# Project Overview

## Название

**Вылазка №1: Периметр**

## Формат

Интерактивная текстовая хоррор-игра для Telegram. Игрок не вводит свободный текст. На каждом сюжетном узле он выбирает один из трёх вариантов ответа:

- 🟦 рациональный;
- 🟥 импульсивный;
- 🟩 эмпатичный.

## Жанр

Психологический хоррор, детектив, научный триллер, интерактивная драма.

## Главная идея

Команда молодых специалистов приезжает в закрытый научный центр «Периметр», чтобы найти пропавшую группу. Постепенно становится ясно: фонд солгал о цели операции, а настоящий ужас связан не с монстрами, а с препаратом **PX-17 / Эхо**, который усиливает память, страх, вину и восприятие.

## Главный конфликт

Правда против людей. Игрок должен решить, что важнее: раскрыть миру правду о PX-17 или спасти тех, кто рядом.

## Атмосфера

- дождь;
- бетон;
- холодный свет;
- сломанные динамики;
- пустые коридоры;
- голоса без источника;
- документы, которые отвечают на один вопрос и создают два новых.

## Структура

История длится три дня:

1. **День 1 — Вход.** Команда знакомится, входит в комплекс и находит первые следы лжи.
2. **День 2 — Распад.** Пропадает Глеб, команда раскалывается, открывается правда об «Эхо».
3. **День 3 — Цена выбора.** Все сюжетные линии сходятся в лаборатории PX-17. Игрок принимает финальное моральное решение.
