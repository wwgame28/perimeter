# Characters

## Главный герой

Облик главного героя намеренно не фиксируется. В визуалах он изображается как чёрный силуэт, чтобы любым игроком мог быть любой пользователь.

Роль героя раскрывается через решения:

- рациональный лидер;
- импульсивный спасатель;
- эмпатичный посредник;
- человек, который выбирает правду любой ценой;
- человек, который выбирает людей вместо правды.
